Requisitos:

1 - Deverá consumir a API contendo o câmbio de Dólar e Real no endereço: https://economia.awesomeapi.com.br/json/last/USD-BRL ok
2 - Deverá retornar no formato JSON o resultado para o cliente. ok
3 - Usando o package "context", o server.go deverá registrar no banco de dados SQLite cada cotação recebida. Ok
4 - timeout máximo para chamar a API de cotação do dólar deverá ser de 200ms. ok
5 - E o timeout máximo para conseguir persistir os dados no banco deverá ser de 10ms. ok
6 - O client.go precisará receber do server.go apenas o valor atual do câmbio (campo "bid" do JSON). ok
7 - Utilizando o package "context", o client.go terá um timeout máximo de 300ms para receber o resultado do server.go. ok
8 - Os 3 contextos deverão retornar erro nos logs caso o tempo de execução seja insuficiente. ok
9 - O endpoint necessário gerado pelo server.go para este desafio será: /cotacao e a porta a ser utilizada pelo servidor HTTP será a 8080. ok
10 - O client.go terá que salvar a cotação atual em um arquivo "cotacao.txt" no formato: Dólar: {valor}
